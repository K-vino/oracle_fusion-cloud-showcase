# Oracle Fusion Cloud Showcase

A minimalist, modern website showcasing our Oracle Fusion Cloud group project with functional and technical implementation details.

## Overview

This project is a static website built with clean HTML, CSS, and minimal JavaScript to showcase a comprehensive Oracle Fusion Cloud group project. The design emphasizes clarity, typography, and user experience with a premium, calm aesthetic.

## Project Structure

```
oracle_fusion-cloud-showcase/
├── index.html          # Main HTML structure
├── style.css           # Minimalist styles with neutral color palette
├── script.js           # Smooth scroll reveal animations
├── README.md           # Project documentation
└── LICENSE            # Project license
```

## Features

### Design Principles
- **Minimalist Aesthetic**: Clean layout with generous white space
- **Neutral Color Palette**: Black, white, and soft gray tones
- **Typography-Focused**: Clear hierarchy and readable fonts
- **Smooth Animations**: Scroll-based reveal using Intersection Observer API
- **Fully Responsive**: Optimized for all device sizes
- **Performance-Focused**: Lightweight and fast-loading
- **Accessible**: Respects prefers-reduced-motion and keyboard navigation

### Website Sections

1. **Hero Section**
   - Large, confident headline
   - Subtle fade-in animation on page load
   - Clean typography with generous spacing

2. **Project Section**
   - Overview of the Oracle Fusion Cloud project
   - Two main tracks:
     - **Functional Track**: Business processes, configuration, workflow mapping
     - **Technical Track**: System architecture, integrations, data flow

3. **Team Section**

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Modern styling with Grid and Flexbox
- **Vanilla JavaScript**: Intersection Observer API for scroll animations
- **No Dependencies**: Pure web technologies, no frameworks required

## Getting Started

### Viewing the Website

Simply open `index.html` in a modern web browser:

```bash
# Using Python's built-in server (recommended)
python3 -m http.server 8000

# Or using Node.js http-server
npx http-server

# Then navigate to http://localhost:8000
```

Alternatively, you can open `index.html` directly in your browser.

### Browser Compatibility

- Chrome 58+
- Firefox 55+
- Safari 12.1+
- Edge 79+

All modern browsers with Intersection Observer API support.

## Development

The website is built with vanilla technologies and requires no build process. All files are ready to deploy as-is.

### File Details

- **index.html**: Contains the semantic structure with sections for hero, project, team, and closing
- **style.css**: Includes responsive design, animations, and accessibility features
- **script.js**: Implements smooth scroll reveal animations using Intersection Observer API

## Performance & Accessibility

- **Lightweight**: No external dependencies or heavy libraries
- **Fast Load Times**: Minimal file sizes and optimized resources
- **Accessible**: Semantic HTML, keyboard navigation support, and reduced motion support
- **SEO-Friendly**: Proper meta tags and semantic structure
